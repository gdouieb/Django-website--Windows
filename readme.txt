anomaPro : anomalies dans la ville de Paris. 

Instructions : 

1) Installer les modules requis pour cette application: 

    executez la commande suivante depuis le dossier anomaPro : 
    pip install -r requirements.txt (ou pip3 install -r requirements.txt)
    
2) verifier que le fichier dans-ma-rue.csv se trouve bien dans :

    .../Anomalies/AnomaliesParis/anomaPro/dans-ma-rue.csv

3) Lancer le serveur django:

    executez la commande suivante depuis le dossier anomaPro :
    python manage.py runserver (ou python3 selon votre installation)
